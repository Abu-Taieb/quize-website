What is the difference between filter() , find(), map() & forEach()

1. Description of filter() : If I want to take a number value from an array, the filter will give me as many values ​​of the number as are the same inside that array. Exm: array = [3, 5, 8, 3, 6, 9]; return [3, 3]


2. Description of find() : If I want to take a number value from an array, the find will give me a first matching value ​​of the number as are the same inside that array. Exm: array = [3, 5, 8, 3, 6, 9]; return [3]


3. Description of map() : map() is used to transform each element in an array. Exm: array = [3, 5, 8, 3, 6, 9]; array.map() return [3, 5, 8, 3, 6, 9]


4. Description of forEach() : forEach() is used to perform a certain operation on each element in an array. Exm: array = [3, 5, 8, 3, 6, 9]; array.forEach() but no return data.

What is the difference between localStorage , sessionStorage , cookies

1. Description of localStorage : If I close my browser, website & laptop but data not erase from localStorage.


2. Description of sessionStorage : If I close my browser, website or laptop then data is gon from sessionStorage. Summary is localStorage and sessionStorage are same but sessionStorage data erase when session is out but if my session is out not erase data from my localStorage.


2. Description of Cookie : Cookie allow you to manage any information that the server should remember. Exm: login info, shopping info, website visit info etc.
